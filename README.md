## webodf
webodf texteditor

## 作用
用于显示odt，odf文件并可在线编辑

## 插件来源
https://webodf.org/

## 执行及访问
执行 node server.js

http://localhost:9526/texteditor.html

http://localhost:9526/localeditor.html

http://localhost:9526/revieweditor.html
